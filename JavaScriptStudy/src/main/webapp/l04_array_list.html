길이와 index 를 사용하는 자료형 Array

js는 java 의 Array 가 없고 ArrayList 와 거의 동일한 자료가 있다.

Object 와 Array 의 차이

  • Array(배열) 는 자료의 순서인 index 와 길이가 무조건 존재한다.
  • Object 는 key 와 value 로 이루어진 자료 구조 (==java.Map)
  • Object 는 인덱스와 길이가 존재하지 않는다.
  • Array 의 index 를 키로 사용하지만 물리적으로 존재하지 않느 순서다.
  • js 에는 key 가 순서가 있고 길이가 있는 Object 를 Array 로 변환하는 함수가 있다.

배열의 필드 접근법

  • list.0 (x)
  • list[0] (o)
  • list.get(index) (X java 에서 가능)

Array 유용함 함수와 필드

  • length
  • Array.prototype.push(items) : 인수로 전달받은 모든 값을 원본 배열의 마지막요소로 추가하고 변경된 length프로퍼티를 반환
  • Array.prototype.pop() : 원본배열에서 마지막요소를 제거하고 제거한 요소를 반환
  • Array.prototype.shift() : 원본배열에서 첫번째 요소를 제거하고 제거한 요소를 반환
  • Array.prototype.unshift(items) : 인수로 전달받은 모든 값을 원본 배열의 선두에 요소로 추가하고 변경된 length 프로퍼티값을 반환
  • Array.isArray(items) : 전달된 인수가 배열이면 true, 아니면 false를 반환
  • Array.prototype.concat(arr,item) : 인수로 전달된 값을 원본 배열의 마지막 요소로 추가한 새로운 배열을 반환. Array.prototype.concat(arr) : 배열뒤에 배열추가하여 반환 /Array.prototype.concat(item) : 마지막요소로 추가후 반환
  • Array.prototype.splice(startIndex,deleteCount,items) : 중간에 있는 요소를 추가하거나 제거 새로운 배열을 반환(원본배열유지)
  • Array.prototype.slice(start,end) : 원본 배열을 변경한다.
  • Array.prototype.join(구분자) : 원본배열의 모든 요소를 문자열로 변환한후 인수로 받은 문자열을 구분자로 연결한 문자열을 반환(default값 :,)
  • Array.prototype.reverse() : 원본 배열의 순서를 반대로 뒤집는다.
  • Array.prototype.fill(item,index(0부터 index까지) : 인수로 전달받은 값을 배열의 처음부터 끝까지 요소(item)로 채운다.
  • Array.prototype.includes(item,index) : 특정요소가 포함되어있는지 확인하여 true, false 를 반환
  • Array.prototype.flat(인수_기본값1) : 인수로 전달한 깊이만큼 재귀적으로 배열을 평탄화 한다.
