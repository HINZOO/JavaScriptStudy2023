변수와 상수에 대해 배워보자

js의 변수와 상수는 타입을 명시하지 않는다 (모든 타입의 데이터를 참조할 수 있다.)

변수 var let

  • js의 변수는 타입을 명시하지 않는다.
  • 변수는 변할 수 있는 수로 참조하는 데이터를 바꿀 수 있다.
  • var : 윈도우 필드로 정의되고 전역에만 선언 가능하다.
  • let : scope(=영역={}블럭)가 명확한 변수로 es6문법이다.

상수와 const

처음 선언하면 참조하는 data를 바꿀 수 없는 것을 상수라고 한다. es6에서 추가되었다. (보통 자료형이나, 함수를 선언할 때 사용한다.) 상수의 예) Math.PI : 변하지 않는 정보를 표기할 때 사용
