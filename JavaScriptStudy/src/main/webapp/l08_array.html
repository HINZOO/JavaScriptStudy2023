배열의 내부 반복문과 정렬

📂 Array Iteration

  • forEach(): 각 배열요소에 대해 함수(콜백함수)를 한번 호출 합니다.(value,index,array의 인수를 받는다)
  • map()
    • 각 배열의 요소에서 함수를 수행하여 새 배열을 만든다.(조건에 맞는 return값으로 새 배열을 반환)
    • 값이 없는 배열요소에 대한 함수를 실행하지 않는다.
    • 원래 배열을 변경하지 않는다.
  • filter() : 조건에 맞지 않는 것을 제외하고 배열로 반환한다.
  • reduce() : 각 배열 요소에서 함수를 실행하여(왼->오 순서로 작동) 단일값을 생성하며, 원래 배열을 줄이지 않는다.
  • every() : 모든 배열의 요소가 조건에 만족하는지 boolean값으로 반환(&&연산)
  • some() : 하나라도 조건에 만족하면 true (||연산)
  • indexOf() : 배열에서 요소값을 검색하고 index값을 반환
  • lastIndexOf() : 지정된 요소의 마지막 발생 위치를 반환
  • find() : 조건의 값을 찾음
  • findIndex() : 조건에 맞는 값의 index를 반환
  • ✨ Array.from() : 유사배열을 배열로 바꿔줌
  • keys() : js에는 물리적으로 key가 없지만 iterator가 index를 key처럼 반환한다.
  • includes() : 값이 포함되었는지 boolean 값으로 반환
