🎀 node 생성과 추가하기

html 문서가 화면에 그대로 출력되고 있다는 착각을 많이 하는데.. 실제 화면에 출력하고 있는 것은 node 개체이다.

📂 document.createElement("TagName"); : 요소노드 개체를 생성하는 함수

📂 document.createTextNode("string"); : 텍스트노드 개체를 생성하는 함수

📂 노드 개체를 추가해주는 함수

  • node.append(node) : 첫번째 자식노드의 맨뒤에 추가
  • node.prepend(node) : 첫번째 자식노드의 맨 앞에 추가
  • node.after(node) : 형제노드의 바로 뒤에 추가
  • node.before(node) : 형제노드의 바로 앞에 추가
  • node.replaceWith(node) : 해당노드를 대체
